# 楽天ROOM 自動投稿システム

楽天ROOMの投稿を、Claude Code がペルソナに沿って自動生成・自動投稿するシステムです。
週次でデータを分析して、自分自身を改善していきます。

## できること

- 1日3回（朝・昼・夜）、ペルソナに沿ったコメント付きで自動投稿
- 楽天ジャンルランキングから商品を選定
- いいね・フォローで露出獲得
- 週次でアフィリエイトレポートを取得・分析
- 分析結果に基づいて投稿戦略を自動更新（コメント長さ、絵文字、価格帯、投稿時間など）

## 動作環境

- macOS（Apple Silicon / Intel どちらも）
- Node.js 20 以上
- Claude Code（**Max プラン強く推奨**）

> **重要**: このシステムは Claude Code のサブスクリプション契約が必須です。
> サブスク契約がないと、コメント生成や週次改善が動きません。
> APIキー方式（従量課金）には対応していません。
>
> **プラン選びについて**:
> - **Max プラン推奨（月額 $100〜）**: 本システムは長時間バックグラウンドで動くため Max プランが安定動作します
> - Pro プラン（月額 $20）でも現在は動作しますが、Anthropic 社が Pro プランからの Claude Code 制限を検討中（2026年4月時点で2%のユーザーで実験中）。**将来的に Pro では使えなくなるリスクがあります**。長期運用するなら Max を推奨します

## セットアップ

セットアップ手順は **購入元の Note 記事**に詳しく書かれています。
Mac のターミナル操作に慣れていなくても、コマンドをコピペしながら30〜60分でセットアップできます。

セットアップが終わったら、以降は **Mac を起動するだけ** で
裏で自動投稿・自動改善が動き続けます。
